Program: cppDocumentationMaker

Description: Program that creates an HTML file with the documentation of a CPP file

Author: Juan Francisco Barr√≥n Camacho

Date: 23 / January / 2018

Function: createHTML

Description: Creates the HTML file with the CPP file's name and the first basic lines (head, title, body)

Parameters: filename nname of the cpp file from which the html file will be created

Function: editHTML

Description: Add the documentation using the tag to create the corresponding format (h1, p, etc)

Parameters: tag determines the documentation tag. documentation is the documentation per se. filename is the name of the HTML file to be opened and edited

Function: tagSearch

Description: Looks for a documentation tag in a given line of the CPP file

Parameters: line is the content of a line in the CPP file. filename is the file's name to be sent to editHTML

Function: main

Description: Asks for the name of a CPP file (add the extension if omitted by the user) and creates an HTML file with the documentation
